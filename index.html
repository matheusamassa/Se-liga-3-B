O que aprendi com o projeto

Com este projeto, aprendi como estruturar um site básico usando HTML e CSS. O HTML foi usado para organizar o conteúdo e o CSS para estilizar a página. O uso de tags semânticas como <header>, <main> e <footer> melhora a legibilidade do código e facilita a manutenção do site. Além disso, aprendi a usar classes e IDs para aplicar estilos específicos a determinados elementos.

Explicação do Código

Abaixo, explico o que cada parte do código HTML e CSS faz:

Código HTML:

  • <!DOCTYPE html>: Declara que o documento está usando HTML5.
  • <html lang="pt-br">: Define o idioma da página como português do Brasil.
  • <head>: Contém informações sobre a página, como o título e o link para o arquivo CSS.
  • <body>: Contém o conteúdo visível da página. Aqui, organizamos o cabeçalho, o corpo e o rodapé.
  • <header>: A primeira seção da página, onde colocamos o título da escola e o nome do aluno.
  • <main>: O conteúdo principal, dividido em seções, como informações do aluno e a explicação do que foi aprendido.
  • <footer>: A seção de rodapé, onde incluímos o copyright.
  • <ul> e <li>: Usados para criar uma lista de informações, como nome do aluno e data de entrega.

Código CSS:

  • body {}: Define a fonte, cor de fundo, e remove margens e preenchimentos padrão do navegador.
  • header {}: Aplica estilo ao cabeçalho, como fundo escuro, texto branco, centralização e preenchimento.
  • #informacoes {}: Estilo da seção que contém as informações do aluno, com fundo branco, bordas arredondadas, e preenchimento.
  • #explicacao {}: Estilo para a seção de explicação do projeto, com fundo cinza claro e bordas arredondadas.
  • footer {}: Aplica estilo ao rodapé, tornando-o fixo na parte inferior da tela, com fundo escuro e texto branco.

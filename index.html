O que Aprendi com este Projeto

Com este projeto, aprendi como estruturar um site básico utilizando HTML para definir os elementos e CSS para aplicar estilos visuais. Com a separação do conteúdo (HTML) e da apresentação (CSS), pude entender a importância de organizar o código de forma eficiente, além de aprender como utilizar classes e IDs para aplicar estilos personalizados. O aprendizado inclui como modificar cores, fontes, margens e preenchimentos, o que permite criar sites mais organizados e visualmente atraentes. A utilização de tags semânticas como <header>, <footer>, e <nav> também facilita a compreensão e a manutenção do código.
